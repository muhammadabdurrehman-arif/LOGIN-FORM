# 🔐 Login System

A responsive and user-friendly Login System built using HTML, CSS, and JavaScript. This project demonstrates front-end authentication concepts, form validation, and modern UI design principles.

## ✨ Features

* User Login Interface
* Responsive Design
* Form Validation
* Show/Hide Password Functionality
* Clean and Modern User Interface
* Mobile-Friendly Layout
* Error Message Handling
* Easy Customization

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Login-System/
│
├── index.html

├── script.js
└── README.md
```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/Login-System.git
```

### Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Launch `index.html` in your preferred web browser.

## 📋 Login Functionality

* Validates user input fields.
* Checks username and password requirements.
* Displays error messages for invalid input.
* Allows users to toggle password visibility.
* Provides a smooth user experience.
